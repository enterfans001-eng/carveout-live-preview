# CARVEOUT WordPress Theme

このフォルダは、現在の静的サイトをWordPressで表示するためのテーマ一式です。

## 使い方

1. `carveout-wordpress-theme.zip` をWordPress管理画面からアップロードします。
2. WordPress管理画面で `外観 > テーマ > 新規追加 > テーマのアップロード` を開きます。
3. ZIPを選択してインストールし、有効化します。
4. `固定ページ` で下記スラッグのページを作成します。

## 作成する固定ページ

- `about`
- `about-detail`
- `audition`
- `benefit`
- `compliance`
- `contact`
- `events`
- `interview`
- `interview-detail`
- `kabuu`
- `livers`
- `message`
- `news`
- `news-detail`
- `next-stage`
- `partner`
- `privacy`
- `services`

トップページは `front-page.php` が使われます。

## 注意

この版は、まず現在のデザインをWordPress上で再現するためのテーマです。
NEWS、EVENT、LIVERなどを管理画面から個別編集する完全CMS化は次の段階で対応できます。
