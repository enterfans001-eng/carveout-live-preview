# CARVEOUT WordPress Theme

このフォルダは、現在の静的サイトをWordPressで表示するためのテーマ一式です。

## 使い方

1. `carveout-wordpress-theme.zip` をWordPress管理画面からアップロードします。
2. WordPress管理画面で `外観 > テーマ > 新規追加 > テーマのアップロード` を開きます。
3. ZIPを選択してインストールし、有効化します。
4. `固定ページ` で下記スラッグのページを作成します。
5. `設定 > パーマリンク` を開き、何も変更せずに `変更を保存` を押します。

## 作成する固定ページ

固定ページは、サイトの「箱」です。本文は空でも大丈夫です。
ページタイトルは日本語でも英語でも問題ありませんが、URLスラッグだけ下記に合わせます。

- `about`
- `about-detail`
- `audition`
- `benefit`
- `compliance`
- `contact`
- `events`
- `interview`
- `interview-detail`
- `kabuu`
- `livers`
- `message`
- `news`
- `news-detail`
- `next-stage`
- `partner`
- `privacy`
- `services`

トップページは `front-page.php` が使われます。

## 管理画面から追加できるもの

テーマを有効化すると、管理画面の左メニューに下記が追加されます。

- ニュース
- 事務所イベント
- インタビュー
- 所属ライバー
- ランキング

これはWordPress標準の「投稿」とは別の、CARVEOUT用の投稿欄です。
ニュースは `ニュース`、イベントは `事務所イベント` から追加すると、トップページや一覧ページに反映されます。

### ニュース・事務所イベント・インタビュー

1. 左メニューから該当メニューを開きます。
2. `新規追加` を押します。
3. タイトル、本文、アイキャッチ画像を入れて公開します。
4. 必要に応じて `CARVEOUT表示設定` の `元記事URL` または `画像URL` を入力します。

入力した投稿が1件以上ある場合、一覧ページやトップページではWordPress投稿が優先表示されます。
未入力の場合は、既存の固定データが表示されます。

各欄の使い方:

- タイトル: カードや詳細ページに表示されるタイトル
- 本文: 詳細ページに表示される本文
- アイキャッチ画像: カードや詳細ページの画像
- 元記事URL: 参考元URLを残したい場合のみ入力
- 画像URL: アイキャッチ画像を使わず、外部画像URLを直接使いたい場合のみ入力

### 所属ライバー

1. 左メニューの `所属ライバー` から `新規追加` を押します。
2. タイトルにライバー名を入力します。
3. アイキャッチ画像を設定します。
4. `CARVEOUT表示設定` に配信アプリ名、配信アカウントURL、Instagram URLを入力します。

配信アプリ名の例:

- `17LIVE`
- `BIGOLIVE`
- `TikTokLIVE`
- `Pococha`

### ランキング

1. 左メニューの `ランキング` から `新規追加` を押します。
2. タイトルにライバー名を入力します。
3. アイキャッチ画像を設定します。
4. `CARVEOUT表示設定` に配信アプリ名、ランキング種別、順位、配信アカウントURLを入力します。

ランキング種別の例:

- `総合`
- `新人`

ランキングの表示先:

- 配信アプリ名 `17LIVE` + 種別 `総合`: 17LIVE総合ランキング
- 配信アプリ名 `17LIVE` + 種別 `新人`: 17LIVE新人ランキング
- 配信アプリ名 `BIGOLIVE`: BIGOLIVEランキング

## 更新時の注意

テーマを更新したあと、URLがうまく開かない場合は `設定 > パーマリンク > 変更を保存` をもう一度押してください。
